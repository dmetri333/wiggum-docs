Database: Updates

Simple Update

The update method, like the insert method, accepts an array of column and value pairs containing the columns to be updated. You may constrain the update query using where clauses: 

$this->db->table('users')
    ->update(['votes' => 1])
    ->where('id', 1)
    ->execute();
